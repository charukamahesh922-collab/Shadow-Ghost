# 👻 ShadowGhost - Advanced Ethical Reconnaissance Tool

*"For those who climb where others can't"*

## 🎯 Features

- 🌐 DNS Reconnaissance (A, AAAA, MX, NS, TXT records)
- 🔌 Port Scanning (100+ common ports with banner grabbing)
- 🖥️ Technology Detection (CMS, frameworks, web servers)
- 🛡️ Vulnerability Scanning (SQL injection, XSS, LFI)
- 📂 Directory Discovery (admin panels, sensitive directories)
- 🌍 Hosting Analysis (CDN detection, provider identification)
- 📊 Comprehensive Reports (HTML, JSON, Markdown)
- ⚡ Multi-threaded for fast scanning
- 🔄 Auto-update from GitHub
